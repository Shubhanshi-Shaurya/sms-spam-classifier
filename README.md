# Email/SMS Spam Classifier

An AI-powered Email Spam Classifier that detects whether an email or message is Spam or Not Spam (Ham) using **Machine Learning** and **Natural Language Processing (NLP)**.

---

##  Live Demo
You can access the live web application here: [Email/SMS Spam Classifier](https://sms-spam-classifier-65h9.onrender.com)

---

##  Features
- **Classification:** Classifies emails/messages as Spam or Ham(Not Spam).
- **Modern UI/UX:** Clean User-friendly interface using Streamlit.
- **Natural Language Processing (NLP):** Text preprocessing using NLP techniques.
- **Machine Learning Model:**Machine Learning model training and prediction.


---
 
##  Tech Stack
- **Language:** Python 
- **Data Science:** NumPy, Pandas
- **ML Stack:** Scikit-Learn, Vectorizer, NLTK
- **Framework:** Streamlit
- **Environment Management:** Virtualenv, VS Code
